 Basic Operations

- Arithmetic Operations:
  - What it is: Basic mathematical operations.
  - When it's used: To perform calculations.
  - How it's used:
  python
  a = 5 + 3
  b = 5 - 3
  c = 5 * 3
  d = 5 / 3
  e = 5 % 3
  f = 5 ** 3
  g = 5 // 3

Arithmetic operators are used to perform mathematical operations such as addition, subtraction, multiplication, and division. These operators are essential for handling numerical data in various computations.

#### Types of Arithmetic Operators

1. Addition (+)
   - What it is: Adds two operands.
   - When it's used: To sum up numbers or concatenate strings.
   - How it's used:
     python
     a = 5
     b = 3
     result = a + b
     print(result)  # Output: 8
     

2. Subtraction (-)
   - What it is: Subtracts the right operand from the left operand.
   - When it's used: To find the difference between two numbers.
   - How it's used:
     python
     a = 5
     b = 3
     result = a - b
     print(result)  # Output: 2
     

3.Multiplication (*)
   - What it is: Multiplies two operands.
   - When it's used: To find the product of two numbers.
   - How it's used:
     python
     a = 5
     b = 3
     result = a * b
     print(result)  # Output: 15
     

4. Division (/)
   - What it is: Divides the left operand by the right operand. Returns a float.
   - When it's used: To divide one number by another.
   - How it's used:
     python
     a = 5
     b = 2
     result = a / b
     print(result)  # Output: 2.5
     

5. Floor Division (//)
   - What it is: Divides the left operand by the right operand and returns the largest integer less than or equal to the result.
   - When it's used: To perform division and obtain an integer result.
   - How it's used:
     python
     a = 5
     b = 2
     result = a // b
     print(result)  # Output: 2
     

6. Modulus (%)
   - What it is: Divides the left operand by the right operand and returns the remainder.
   - When it's used: To find the remainder of a division operation.
   - How it's used:
     python
     a = 5
     b = 3
     result = a % b
     print(result)  # Output: 2
     

7. Exponentiation (**)
   - What it is: Raises the left operand to the power of the right operand.
   - When it's used: To perform power calculations.
   - How it's used:
     python
     a = 2
     b = 3
     result = a ** b
     print(result)  # Output: 8

Arithmetic operators are fundamental tools in Python programming, 
allowing you to perform a wide range of mathematical operations.

- Comparison Operators:
  - What it is: Operators that compare values.
  - When it's used: To compare data and make decisions.
  - How it's used:
  python
  x == y
  x != y
  x > y
  x < y
  x >= y
  x <= y

Comparison operators are used to compare two values or variables.
 They return either True or False based on the condition. 
These operators are essential for decision-making in programming, 
often used in conditional statements and loops.

#### Types of Comparison Operators

1. Equal to (==)
   - What it is: Checks if the values of two operands are equal.
   - When it's used: To verify if two values are identical.
   - How it's used:
     python
     a = 5
     b = 5
     print(a == b)  # Output: True
     

2. Not equal to (!=)
   - What it is: Checks if the values of two operands are not equal.
   - When it's used: To verify if two values are different.
   - How it's used:
     python
     a = 5
     b = 3
     print(a != b)  # Output: True
     

3. Greater than (>)
   - What it is: Checks if the value of the left operand is greater than the value of the right operand.
   - When it's used: To compare two values when one should be larger.
   - How it's used:
     python
     a = 7
     b = 5
     print(a > b)  # Output: True
     

4. Less than (<)
   - What it is: Checks if the value of the left operand is less than the value of the right operand.
   - When it's used: To compare two values when one should be smaller.
   - How it's used:
     python
     a = 3
     b = 5
     print(a < b)  # Output: True
     

5. Greater than or equal to (>=)
   - What it is: Checks if the value of the left operand is greater than or equal to the value of the right operand.
   - When it's used: To ensure one value is at least as large as another.
   - How it's used:
     python
     a = 5
     b = 5
     print(a >= b)  # Output: True
     a = 6
     print(a >= b)  # Output: True
     

6. Less than or equal to (<=)
   - What it is: Checks if the value of the left operand is less than or equal to the value of the right operand.
   - When it's used: To ensure one value is at most equal to another.
   - How it's used:
     python
     a = 5
     b = 5
     print(a <= b)  # Output: True
     a = 4
     print(a <= b)  # Output: True
     

#### Examples and Use Cases

- Conditional Statements:
  Comparison operators are commonly used in if, elif, and else statements to control the flow of the program.
  python
  age = 18

  if age >= 18:
      print("You are an adult.")
  else:
      print("You are a minor.")
  

- Loops:
  Comparison operators help in determining when to continue or break out of a loop.
  python
  count = 0

  while count < 5:
      print(count)
      count += 1
  

- Functions:
  They are used within functions to compare arguments or conditions.
  python
  def compare_numbers(x, y):
      if x == y:
          return "Numbers are equal."
      elif x > y:
          return "First number is greater."
      else:
          return "Second number is greater."

  print(compare_numbers(10, 5))  # Output: First number is greater.
  

- Sorting and Filtering:
  When sorting or filtering data, comparison operators play a crucial role.
  python
  numbers = [1, 3, 5, 2, 4, 6]

  even_numbers = [num for num in numbers if num % 2 == 0]
  print(even_numbers)  # Output: [2, 4, 6]
  

Comparison operators are fundamental tools in Python programming, 
enabling complex decision-making and control over the program's logic. 
Understanding and using them effectively is crucial for any Python developer.
  

- Logical Operators:
  - What it is: Operators that combine multiple conditions.
  - When it's used: To perform logical operations.
  - How it's used:
  python
  a and b
  a or b
  not a

Logical operators are used to combine conditional statements. They include and, or, and not. These operators are essential for creating complex conditions in your code.

#### Types of Logical Operators

1. and
   - What it is: Returns True if both operands are true.
   - When it's used: To ensure multiple conditions are met.
   - How it's used:
     python
     a = True
     b = True
     result = a and b
     print(result)  # Output: True

     a = True
     b = False
     result = a and b
     print(result)  # Output: False
     

2. or
   - What it is: Returns True if at least one of the operands is true.
   - When it's used: To check if at least one condition is met.
   - How it's used:
     python
     a = True
     b = False
     result = a or b
     print(result)  # Output: True

     a = False
     b = False
     result = a or b
     print(result)  # Output: False
     

3. not
   - What it is: Returns True if the operand is false, and False if the operand is true.
   - When it's used: To invert the truth value of a condition.
   - How it's used:
     python
     a = True
     result = not a
     print(result)  # Output: False

     a = False
     result = not a
     print(result)  # Output: True
     

#### Examples and Use Cases

- Combining Conditions:
  Logical operators are used to combine multiple conditions in if statements.
  python
  age = 20
  has_ticket = True

  if age >= 18 and has_ticket:
      print("You can enter the concert.")
  else:
      print("You cannot enter the concert.")
  

- Multiple or Conditions:
  Logical operators allow for checking multiple conditions where at least one must be true.
  python
  has_permission = False
  knows_password = True

  if has_permission or knows_password:
      print("Access granted.")
  else:
      print("Access denied.")
  

- *Using not to Invert Conditions*:
  The not operator can be used to simplify conditions by inverting their truth values.
  python
  is_logged_in = False

  if not is_logged_in:
      print("Please log in.")
  else:
      print("Welcome!")
  

- Nested Logical Conditions:
  Logical operators can be nested to form more complex conditions.
  python
  age = 25
  is_employee = True
  has_access_card = False

  if (age > 21 and is_employee) or has_access_card:
      print("Access granted.")
  else:
      print("Access denied.")
  

- Short-Circuit Evaluation:
  Python uses short-circuit evaluation for logical operators,
 meaning it stops evaluating as soon as the result is determined.
  python
  def print_and_return_true():
      print("Function called")
      return True

  a = False
  b = print_and_return_true()

  if a and b:
      print("Both conditions are true.")
  # Output: No output from the print statement inside the if block
  # because the first condition (a) is False, and b is never evaluated.
  

Logical operators are fundamental for creating 
flexible and powerful conditional logic in Python. 
Understanding how to use them effectively allows you to handle complex 
decision-making processes in your programs.
  

6. Control Structures

- Conditional Statements:
  - What it is: Statements that execute code based on conditions.
  - When it's used: To make decisions in code.
  - How it's used:
  python
  if condition:
      # code to execute if condition is true
  elif another_condition:
      # code to execute if another_condition is true
  else:
      # code to execute if none of the above conditions are true
  

- Loops:
  - What it is: Constructs that repeat a block of code.
  - When it's used: To perform repetitive tasks.
  - How it's used:
  python
  for i in range(5):
      print(i)

  count = 0
  while count < 5:
      print(count)
      count += 1
  

- Loop Control Statements:
  - What it is: Statements that alter loop execution.
  - When it's used: To control the flow within loops.
  - How it's used:
  python
  for i in range(5):
      if i == 3:
          break
      print(i)

  for i in range(5):
      if i == 3:
          continue
      print(i)

  for i in range(5):
      if i == 3:
          pass
      print(i)
